# IPL Data Analysis and Machine Learning

Beginner Data Science Project using IPL datasets (2008–2022).

---

## Project Overview

This project analyzes IPL match data using:
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Machine Learning Models

The project uses Logistic Regression and Random Forest models to analyze match outcomes.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
